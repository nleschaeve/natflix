## Quelles sont les trois couches : 
### Frontend : 
Son rôle est d'afficher l'interface et gérer les interactions utilisateur
Les technologies utilisées les plus courantes sont HTML, CSS et JavaScript

### Backend
Son rôle est de gérer les traitements (authentification, recommandations, permissions... )
de faire le lien entre le frontend et la base de données

### Base de données
Son rôle est d'enregistrer, de modifier et de fournir les données
Les technologies utilisées les plus courantes sont MySQL, PostgreSQL ou MongoDB

## Description de ce qui peut être mis sur une page d'accueil
### En tête (Header)
Il contient le logo du site
C'est le haut de la page, qui reste souvent visible lors de la navigation. Il peut contenir les boutons d'interration pour revenir à la page d'accueil de l'application
Il peut donner des information sur l'utilisateur

### Zone de mise en avant
Un caroussel qui présente les films/séries du moment

### Contenu 
Une liste infinie de films populaires ou récents

### Pied de page (Footer)
Contient les mentions légales, les conditions d'utilisation, la politique de confidentialité et des liens vers les réseaux sociaux ou l'aide.
